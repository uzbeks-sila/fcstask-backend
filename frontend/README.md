# FCS Task Frontend

React + TypeScript app built with Vite.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Docker

Build and run:

```bash
docker build -t fcs-task-frontend .
docker run --rm -p 8080:80 fcs-task-frontend
```

Then open http://localhost:8080
